## index.html の変更内容

### 設計方針
- 既存の暫定デザインから実質的なリニューアル
- 「ポーカー情報サイト」ではなく「ポーカーと旅を接続する個人ブランドの公開ページ」として見えることを重視
- `poker-trail_brand.md` の理念・キャッチコピー・世界観を全面に反映

### セクション構成（新）

| セクション | 内容 |
|---|---|
| グローバルナビ | 固定表示、スクロール追従 |
| ヒーロー | 全画面ダーク背景、ロゴ・キャッチフレーズ・ブランドサブコピー |
| About | ブランドステートメント（何の場所かが伝わる文脈） |
| 旅のログ | 記事カード（記事ページへの導線） |
| 活動内容 | 3つのサービス（簡潔に整理） |
| 理念 | 引用形式＋3つのフィロソフィポイント |
| プロフィール | 写真＋テキスト2カラムレイアウト |
| Contact | Googleフォーム・Xリンク |
| フッター | コピーライト |

### デザイン変更点

- **カラー：** warm off-white（#f5f4f1）+ ダーク（#1a1a1a）+ ゴールド（#c8a84b）に統一
- **タイポグラフィ：** Montserrat（英字ラベル・数字）+ Noto Sans JP（本文）の組み合わせを維持・整理
- **ナビ：** 固定ナビを新設（backdrop-filter: blur付き）
- **ヒーロー：** full-height のダーク背景、ブランドの世界観を最初に伝える構造に
- **旅のログカード：** 画像＋テキストの2カラムカード形式。ホバーでリフトアップ
- **プロフィール：** 左カラムに写真固定（sticky）、右カラムに本文という2カラムレイアウト
