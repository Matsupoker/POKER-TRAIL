## 追記：ヒーロー画像組み込み（2026-03-17）

### 作業概要

`poker-trail_hero_airport_departure.png`（空港出発シーン）をトップページのヒーローセクションにメインビジュアルとして組み込んだ。

### 変更箇所

`index.html` の `.hero` および `.hero::before` CSS のみ修正（HTML 変更なし）。

### 実装内容

**背景画像の設定（`.hero` に追加）：**

```css
background-image: url('poker-trail_hero_airport_departure.png');
background-size: cover;
background-position: center 35%;
```

`background-position: center 35%` とすることで空港内部・搭乗口・歩く人物がバランスよく表示されるよう調整。

**オーバーレイの変更（`.hero::before`）：**

旧：単色の薄い radial-gradient
新：4段階の縦グラデーション

| 位置 | 不透明度 | 意図 |
|---|---|---|
| 上端（0%） | 92% | ロゴ・ラベルの視認性を確保 |
| 中上（42%） | 72% | キャッチフレーズの背後で画像の大気感を最大化 |
| 中下（70%） | 78% | サブコピーの可読性を維持 |
| 下端（100%） | 93% | スクロールインジケーター周辺を締める |

**スタック順の明示（`.hero > *` を追加）：**

```css
.hero > * {
    position: relative;
    z-index: 1;
}
```

overlay（z-index: 0）の上にすべてのコンテンツが確実に重なるよう明示。

### 画像について

空港ターミナル内をスーツケース（スペードマーク入り）を引いて歩く後ろ姿の写真。「旅立ち・前進・trail の始まり」を象徴するビジュアルとして、Poker Trail のブランドイメージと高い親和性を持つ。

## 追記：ヒーロー微調整（2026-03-17 第2回）

### 作業概要

ヒーロー全体の印象を「前向き・旅立ち感」に整えるため、オーバーレイの明るさ調整・テキスト要素の密度軽減・ラベル表記の変更を実施。

### 変更内容

**オーバーレイを軽量化（`.hero::before`）：**

| 位置 | 旧 | 新 |
|---|---|---|
| 上端（0%） | 0.92 | 0.80 |
| 中上（40%） | 0.72 | 0.50 |
| 中下（68%） | 0.78 | 0.58 |
| 下端（100%） | 0.93 | 0.82 |

中央付近を大きく開けることで、空港の明るさ・前向きさが増した。

**ヒーロー要素の並び順変更（HTML）：**

- 旧：ロゴ → ラベル → キャッチコピー → サブコピー → Scroll
- 新：ロゴ → キャッチコピー → サブコピー → ラベル → Scroll

ラベルをキャッチコピーの前から後ろへ移動し、中央の視覚的密度を解消。

**ラベルのスタイル調整（`.hero-label`）：**

| プロパティ | 旧 | 新 |
|---|---|---|
| `font-size` | 0.7rem | 0.6rem |
| `font-weight` | 700 | 600 |
| `letter-spacing` | 4px | 3.5px |
| `color` | `var(--gold)`（不透明） | `rgba(200,168,75, 0.42)` |

**ラベル表記変更：** `松島力也` → `まつ`

**その他余白調整：**

- `.hero-logo` の `margin-bottom`：32px → 44px
- `.hero-sub` の `margin-bottom`：52px → 20px（ラベルが後続するため）

## 追記：ヒーロー文字可読性の微調整（2026-03-17 第3回）

### 作業概要

オーバーレイを軽くした結果、明るい空港背景に対して本文テキストが沈みすぎていたため、文字色と text-shadow を最小限調整。

### 変更内容

**ヒーロー本文（`.hero-sub`）：**

| プロパティ | 旧 | 新 |
|---|---|---|
| `color` | `rgba(255,255,255, 0.42)` | `rgba(255,255,255, 0.72)` |
| `text-shadow` | なし | `0 1px 12px rgba(0,0,0, 0.50)` |

opacity を +0.30 引き上げ。text-shadow は背景が明るい部分でもエッジを際立てる最小限の補助。

**ヒーローラベル（`.hero-label`）：**

| プロパティ | 旧 | 新 |
|---|---|---|
| `color` | `rgba(200,168,75, 0.42)` | `rgba(200,168,75, 0.65)` |
| `text-shadow` | なし | `0 1px 10px rgba(0,0,0, 0.45)` |

ゴールドは白より知覚輝度が低いため、サブコピーと同等の見え方になるよう 0.65 に設定。控えめな補助要素の位置づけは維持。
