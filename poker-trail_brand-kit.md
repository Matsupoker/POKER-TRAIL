## 1. Brand Core

### Brand Name

**Poker Trail**  
表記の基本は英字の `Poker Trail`。ロゴ・ナビゲーション・英字見出しでは `POKER TRAIL` の大文字表記も使用可。

### Tagline

**ポーカーと旅をしよう**

Poker Trail を最も短く説明する中心コピー。  
ポーカーを「勝負」だけでなく、「移動・出会い・体験・記録」として捉えるブランド思想を表す。

### Philosophy

**ポーカーと共に新しい旅を創造し、長く愛される環境を育む**

### Brand Statement

Poker Trail は、ポーカーを単なる遊びや勝負としてだけではなく、**人と場所をつなぐ旅**として捉えるプロジェクトである。  
山口を起点に、ポーカーシーンの記録と育成を続けながら、プレイヤーと店舗の架け橋となること。そして、ポーカーが長く愛される環境を共に育むことを目的とする。

### Brand Keywords

- Poker
- Travel
- Yamaguchi
- Community
- Integrity
- Record
- Local Culture
- Sustainable Growth

## 2. Brand Personality

Poker Trail の人格は、派手な煽りではなく、静かな格好良さを持つ個人ブランドである。

### Personality Traits

| Trait | Direction |
|---|---|
| 知的 | 感情だけでなく、構造・背景・文脈を語る |
| 誠実 | 事実と体験を盛らずに伝える |
| 旅感 | 移動、土地、店、人との出会いを大切にする |
| 落ち着き | 過度にポップにしない。余白と低彩度を活かす |
| ローカル志向 | 山口を起点に、地域のポーカー文化を支える |
| 持続可能性 | 一時的な盛り上がりではなく、長く愛される環境を目指す |

### Tone of Voice

- 敬体・常体のどちらも可。ただし、公式説明では敬体寄りが安全。
- 旅ログでは一人称の体験記として、やや常体寄りの自然な語りも可。
- 「煽る」「盛る」「過剰に成功者っぽく見せる」表現は避ける。
- 文章は落ち着かせるが、体験の熱量は残す。
- ポーカー用語・ビジネス用語は使ってよいが、読者が置いていかれない粒度にする。

### Voice Examples

#### Good

> ポーカーは、ゲームであり、旅でもある。人と場所をつなぐ体験の記録を、ここに積み上げていく。

> 自分の大事な遊び場は、自分で守る。Poker Trail は、そのための小さな実践から始まった。

> 山口を起点に、ポーカーが長く愛される環境を育てていく。

#### Avoid

> 最強のポーカー情報サイト、爆誕。

> 誰でも勝てる、稼げる、人生が変わる。

> 山口ポーカー界を完全制覇する。

## 4. Visual Identity

### Design Direction

Poker Trail のビジュアルは、**黒・金・クリームを基調にした、落ち着いた旅とポーカーの世界観**を基本とする。

- 黒：夜、集中、知性、ポーカーの緊張感
- 金：価値、灯り、特別感、勝負の余韻
- クリーム：旅の記録、紙、温度感、読みやすさ
- 写真：実際の旅・街・店・移動の記録を使う
- 装飾：控えめ。線、タグ、余白、番号を活用する

## 6. Typography

既存サイトの指定に合わせる。

### Primary Japanese Font

**Noto Sans JP**

用途：本文、見出し、説明文、旅ログ本文。  
理由：読みやすく、硬すぎず、公式感と個人感のバランスが良い。

### Primary Latin Font

**Montserrat**

用途：ナビゲーション、英字ラベル、日付、タグ、番号、CTA。  
理由：旅・ブランド・ポーカーのスタイリッシュさを出しやすい。

### Font Stack

```css
font-family: 'Noto Sans JP', 'Montserrat', sans-serif;
```

英字ラベル・ナビ・タグでは以下を優先する。

```css
font-family: 'Montserrat', sans-serif;
```

### Typography Rules

- 英字ラベルは `uppercase`、文字間は広めにする。
- 日本語見出しは太字だが、過度に巨大化しない。
- 本文は行間を広めに取り、旅ログとして読める余白を確保する。
- 数字・日付・セクション番号は Montserrat で統一するとブランド感が出る。

## 7. Logo and Assets

### Current Logo Asset

`Poker Trail ロゴ.png`

### Hero Asset

`poker-trail_hero_airport_departure.png`

### Profile Asset

`自己紹介画像.jpg`

### Logo Usage

- 黒背景・写真背景の上に配置する場合は、周囲に十分な余白を取る。
- ロゴの近くに強い装飾を置きすぎない。
- ヒーローではロゴ単体、キャッチフレーズ、サブコピーの順で見せる。
- ロゴが読みにくい場合は背景側を暗くする。ロゴそのものを過度に加工しない。

### Do

- 黒背景、暗め写真、金色アクセントと組み合わせる。
- 余白を広く取り、静かな存在感を出す。
- ロゴ下に `ポーカーと旅をしよう` を配置する。

### Don’t

- 極端に明るい原色背景に置く。
- ロゴの周囲に情報を詰め込む。
- 影や縁取りを過剰に足してチープにする。
- ポーカーのみ、旅行のみの印象に偏らせる。

## 9. Imagery Guidelines

### Image Direction

Poker Trail の写真・画像は「旅の実在感」を重視する。

- 駅、空港、港、街並み、飲食店、移動風景
- ポーカーに直接関係しない写真も、旅や記録の文脈があれば使用可
- AI生成画像を使う場合も、既存写真の焼き直しではなく、ブランドの世界観を補う用途に留める
- 写真は明るすぎる観光パンフレット風より、少し落ち着いたトーンが合う

### Hero Image Rules

- 写真背景には暗めのオーバーレイを入れ、文字の可読性を最優先する。
- 中央に強い情報が集まりすぎる画像は避ける。
- 「出発」「移動」「旅の始まり」を感じる画像と相性が良い。

### Article Image Rules

- 実際の旅程・店・料理・移動が伝わる写真を使う。
- キャプションを入れ、記録性を高める。
- 写真の枚数が多い場合は、グリッドで整理する。

## 10. Content Pillars

Poker Trail で扱う内容は、以下の柱に整理する。

### 1. Trail Log

旅、移動、食、サウナ、街歩き、ポーカー遠征などの体験記。  
ブランドの世界観を最も具体的に見せる領域。

### 2. Poker Community

山口ポーカー交流チャット、地域プレイヤー、店舗との関係性。  
「人と場所をつなぐ」機能を担う。

### 3. Poker Hub

店舗経営者向けの情報、法規制、運営ノウハウ、イベント管理など。  
ブランドの実務的価値を担う。

### 4. Product / Tool

Heads Up Duel Timer など、ポーカー体験を支えるツール。  
静かな技術力・実装力を示す領域。

### 5. Philosophy / Notes

ポーカー文化、遊び場を守る思想、持続可能な地域コミュニティに関する考察。

## 12. Use Case Guidelines

### Website

- 黒背景＋金アクセント＋クリーム本文で統一する。
- ページ上部では世界観、下部では活動内容・連絡導線を示す。
- 旅ログはブランドの入口として扱う。

### SNS

- 投稿文は短く、煽らない。
- 写真には一言の記録性を持たせる。
- ハッシュタグを盛りすぎない。
- 「旅」「記録」「山口」「ポーカー」の文脈を維持する。

### Note / Blog

- 旅ログは体験記として、時系列・費用・ルート・感想を整理する。
- ただの日記ではなく、「読み手が追体験できる記録」にする。
- 店名・地名・時間・価格などの具体情報を入れると Poker Trail らしさが増す。

### Presentation / Proposal

- ブランド色は黒・金・クリームの範囲で統一する。
- 過度な装飾よりも、余白・写真・短いコピーで見せる。
- 提案資料では「地域ポーカー文化の持続可能性」を軸にすると一貫する。

## 13. Brand Guardrails

### Must Keep

- ポーカーと旅の両方を扱う
- 山口を起点とする
- 個人ブランドとしての誠実さを保つ
- 店舗とプレイヤーの架け橋という視点を持つ
- 派手さより、長く続く信頼感を重視する

### Must Avoid

- ギャンブル的な射幸性を過度に煽る
- 稼げる、勝てる、人生逆転などの誇大表現
- 店舗やプレイヤーを対立的に扱う表現
- 露骨な情報商材風の見せ方
- 既存写真をそのまま焼き直したようなAI画像表現
- ブランドの静かな格好良さを壊す過剰な装飾

## 14. Implementation Notes

### Current Design Strengths

- 黒・金・クリームの配色が既に定まっている
- Montserrat と Noto Sans JP の組み合わせが安定している
- 「旅のログ」と「Poker Hub / 活動内容」が自然につながっている
- 個人ブランドでありながら、過度に自己紹介へ寄りすぎていない

### Next Design Principle

今後ページや記事を追加する場合は、**新しい要素を足す前に、既存の黒・金・クリーム・余白・写真・記録性のルールに沿うかを確認する。**

Poker Trail は、情報量よりも「静かな信頼感」と「旅の余韻」が先に立つブランドとして設計する。
